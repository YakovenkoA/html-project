---------------------блочные элементы---------------
adress - адреса
article - статья
aside - боковые колонки
nav - навигация на странице
main - основное содержимое страницы
header - верхняя часть раздела/страницы
footer - нижняя
section - раздел страницы
audio
video
blockquote - блочная цитата
caption - заголовок таблицы
colgroup - определяет группу столбцов таблицы
table - таблица
thead, tbody, tfoot - выделение групп строк таблицы
ford - создание формы
datalist - контент для заполнения поля формы
select - выпадающий список
fieldset - группа полей формы
legend - заголовок для fieldset
dl, ul, ol - теги списков
canvas - рисование с js
details - интеративный элемент с частично скрытым 
summary - видимая часть текста details
figure - обертка элемента <img>
figcaption - заголовок для тега <figure>
map - интеративная карта
iframe - контейнер дл загрузки содержимого другой html
pre - преформатированный текста
progress - индикатор выполнения
---------------------строчные элементы---------------
a - ссылка
abbr - аббривиатура 
b, strong - жирный
i, em - курсив
dfn - определение
br - перенос строки
button, input, textarea - кнопка, текстовое поле,
текстовая область - элементы формы
label - метка для элемента формы
cite - заголовое работы, ссылка на источник
code - код
kbd - код с клавиатуры
samp - пример ввывода программы
var - переменная
col - столбец в таблице
del - текст, удаленный
ins - текст, добавленный
s - аналог del
span - контейнер для css
sub - нижний индекс
sup - верхний индекс
small - маленький текст
img - вставка изображений
mark - выделенный текст
meter - горизонтальный счетчик
q - короткая цитата
